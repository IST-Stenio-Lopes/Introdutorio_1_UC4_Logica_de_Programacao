Dicionários

As estruturas de dados do tipo dicionário são estruturas que associam um valor a uma determinada chave. A principal diferença, entre os dicionários e as listas, é que estas são ordenadas de uma forma específica e seus elementos são acessados por um índice numérico, enquanto que nos dicionários os elementos possuem uma chave que é utilizada para recuperar um item, não tendo assim uma ordem específica. A vantagem de se utilizar um dicionário é que o processo de recuperação de um item é feito de forma mais eficiente. Nas listas, para se acessar um elemento, é necessário percorrer todos os elementos anteriores a ele. Já nos dicionários, o acesso é feito executando-se uma função matemática sobre a chave e daí obtém-se o valor associado. Por exemplo, em uma lista com um milhão de elementos, você poderá ter que percorrer até um milhão de elementos para acessar um determinado item. Nos dicionários, independentemente da quantidade de elementos, o acesso a um item é feito sempre ao mesmo tempo, otimizando o desempenho. A desvantagem do uso de dicionários é que ocupa-se mais memória: é necessário armazenar a chave e o valor.

Lista Dicionário
Ordenação Dados ordenados Dados não ordenados
Acesso Índice numérico Função matemática sobre a chave
Desempenho Mais lento. Pode ser necessário percorrer vários elementos para encontrar o procurado Mais rápido. Independente da posição, o tempo de acesso é sempre o mesmo
Memória Menor uso de memória - armazena apenas o valor Maior uso de memória - armazena chave e valor
