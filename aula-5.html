Operadores aritméticos

Para realizar cálculos matemáticos na programação, é necessária a utilização de operadores aritméticos. Eles são usados nas expressões de soma, subtração, multiplicação e divisão que todos conhecemos da matemática, porém, alguns possuem suas próprias simbologias, que são compreendidas pelo software quando um cálculo é realizado.

Operador Álgebra comum Linguagens de Programação (C#, Java, Phython)
Adição + +
Subtração - -
Multiplicação X *
Divisão ÷ /

Existem mais dois operadores aritméticos que são bastante usados em programação. O primeiro é para a operação de incremento, que é representado por ++ (dois sinais de adição juntos), e é utilizado para adicionar o valor 1 a uma variável, muito utilizada em algoritmos com algum tipo de contador, por exemplo, quando seu código executasse determinada função, seria adicionado 1 ao contador. Essa seria uma forma de informar o número de vezes que aquela função foi executada.

O segundo operador é da operação de decremento, sendo representado por -- (dois sinais de subtração juntos) e é utilizado para subtrair 1 de uma variável. Imagine que você precisa de um contador que irá diminuindo de 1 em 1 até chegar a zero, aí seria muito mais prático utilizar esse operador.

Incremento e decremento são ações que o computador faz bastante, visto que o controle de execução e o momento de retornar para o usuário ocorre praticamente em todos os aplicativos. Vejamos um exemplo em jogos: Em League of Legends, quando um personagem sofre um dano ao longo do tempo, é preciso um contador para registrar o dano total aplicado. Esse dano total é sempre incrementado à medida que o campeão causa dano em algum inimigo, permitindo que todo o dano aplicado em inimigos seja exibido ao final da partida.

Algumas linguagens de programação (C#, Python, dentre outras), apresentam mais operadores aritméticos, como a operação de resto da divisão, chamado de mod, que é representado por %.
Exemplo: 5%2 = 1
Nesse exemplo lê-se como 5 mod 2 que tem como resultado o valor 1, pois o resto da divisão de 5 por 2 é 1.

Como você já deve ter percebido, os operadores aritméticos são operadores nos quais você já está familiarizado desde criança, sendo utilizados nas mais diversas áreas do conhecimento. No próximo tópico conheceremos operadores que são utilizados quando se deseja comparar expressões lógicas sendo utilizadas na tomada de decisões pelos softwares.

Operadores relacionais

Os operadores relacionais são utilizados para comparar os valores entre expressões lógicas, resultando em resposta verdadeira “V” ou falsa “F”. São utilizados massivamente nos mais diversos tipos de softwares e sua função é realizar checagens e ativar ou não, uma rotina dentro de um programa, caso o valor das expressões estejam dentro das regras do operador escolhido.

Vamos começar pelo operador de Igual a, representado pelo símbolo “ == ” (dois sinais de igualdade juntos), é utilizado para verificar se um valor é igual a outro, independente do seu tipo, retornando verdadeiro se for igual ou falso se for diferente.

Em seguida, há o operador Diferente de, representado pelo simbolo “ != ” (sinal de exclamação junto do sinal de igualdade), é utilizado para verificar se um valor é diferente de outro, retornando verdadeiro se for diferente ou falso se for igual.

Os operadores Maior que e Menor que, representados respectivamente por “ > ” e “ < ”, são utilizados exatamente como na matemática, servem para verificar se um valor é maior ou menor que outro, retornando true ou false.

Há também os operadores Maior ou igual a… e Menor ou igual a…, representados respectivamente por “>= ” e “ <= ”, se assemelhando muito com os vistos anteriormente, esses operadores também checam se um número é maior ou menor que outro, mas agora, também checam se os números são iguais. Nesse caso, a operação resultará em verdadeiro, se pelo menos uma uma das checagens forem atendidas, por exemplo, se pelo menos o número for maior ou for igual retornará verdadeiro.

Operadores Lógicos

Agora, vamos conhecer o conceito de operadores lógicos, que podem variar de simbologia, dependendo da linguagem de programação utilizada, mas representam a mesma coisa. Para esse estudo vamos usar operadores de duas linguagens muito comuns como exemplo, que são o C# e o Java.

Os operadores lógicos têm como o objetivo obter uma resposta ao retornar verdadeiro ou falso, a partir da combinação de uma ou mais entradas em um determinado sistema, ou seja, é utilizado para comparar expressões lógicas ou variáveis booleanas. Ao decorrer do texto vamos perceber que os operadores lógicos são simples de serem compreendidos e aplicados a qualquer lógica de programação.

O operador NOT representado pelo símbolo de exclamação “ ! ”, e é utilizado para inverter o valor booleano, devendo ser escrita antes dele, informando que se a o valor for verdadeiro, passará a ser falso, ou seja, se o valor for true, se tornará false.

O operador AND é representado pelo símbolo comercial duplo “ && “, que é utilizado para comparar dois valores booleanos (que podem ser em forma de expressões ou variáveis), resultando em true (verdadeiro) se ambas as expressões forem verdadeiras, senão retorna false (falso).

Expressões Lógicas

Expressões lógicas utilizam operadores lógicos e os seus operandos são os resultados de operações relacionais com variáveis booleanas.

Uma expressão lógica é utilizada para dar a condição necessária que as estruturas lógicas, dentro de um algoritmo, precisam para executar seus comandos. Sendo o resultado da expressão lógica um único valor lógico, será com base nesse valor que serão executados os comandos. A prioridade entre os operadores numa expressão lógica é:

1- Aritmético (prioridade igual a da matemática)
2- Relacional (ordem de aparição)
3- Operador Lógico NOT
5- Operador Lógico AND
6- Operador Lógico OR

O resultado dessa expressão será sempre um ÚNICO valor lógico (true ou false).

Expressões Aritméticas

As expressões aritméticas são constituídas por operadores aritméticos e variáveis numéricas, basicamente são cálculos matemáticos como já estamos acostumados a fazer, seguindo a mesma ordem de precedência, mas agora, também, usamos variáveis para representar números. Importante frisar que o resultado de uma expressão aritmética será sempre um valor numérico.
